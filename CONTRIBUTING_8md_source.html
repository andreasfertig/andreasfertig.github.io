1 # Contributing to C++ Insights
2 
3 Thanks for considering contributing to **C++ Insights**.
4 
5 **C++ Insights** follows a [Code of Conduct](CODE_OF_CONDUCT.md) which aims to foster an open and welcoming environment.
6 
7 ## In brief
8 
9 C++ Insights is a clang tooling project. Therefore it requires libclang to be installed. You can compile it either with
10 g++ or clang++.
11 
12 The current C++ language version used is C++17.
13 
14 
15 ## Contribute
16 
17 There is a Travis CI running which compiles the code on Linux and macOS. Currently, it is limited to a out-of-source
18 build. Windows is not supported yet. Please check the Travis CI status for each PR. The macOS build runs the unit tests.
19 Code coverage runs on one of the Linux images.
20 
21 ### Fixing an Issue
22 
23 Create a test named: `Issue<ISSUE_NUMBER>.cpp` and a `Issue<ISSUE_NUMBER>.expect` with the transformed code after the fix.
24 
25 The current style in the PR is:
26 ```
27 Fixed #<ISSUE_NUMBER>: <SHORT EXPLANATION>
28 
29 <DETAILD EXPLANATION>
30 ```
