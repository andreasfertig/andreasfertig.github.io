Detailed Description

template<bool VALUE>
class clang::insights::Once< VALUE >

A helper object which returns a boolean value just once and toggles it after the first query.

This allows to simplify code like this:

bool first{true};
for(... : ...) {
if(first) {
first = false;
...
} else {
...
}

into this:

OnceTrue first{};
for(... : ...) {
if(first) {
...
} else {
...
}
