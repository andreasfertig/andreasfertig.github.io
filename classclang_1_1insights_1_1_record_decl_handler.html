Detailed Description

Show a struct or class with the special members the compiler generates.

Conveniently the compiler generates special member for us. Like the default constructor, copy or move operators. Under certain circumstances it stops generating. This handler rewrites an entire class and shows all the pieces the compiler adds for us.
