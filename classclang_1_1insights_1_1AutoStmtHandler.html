Detailed Description

Show the types behind auto.

The AutoStmtHandler matches all auto statements in global space. That means:

auto i = 0;
auto [x,y] = Point{1,2};
auto lambda = [&]() { return errno; };

The transformation itself shows the fully expanded type behind auto and a possible associated statmente like lambda or structured binding. For example:

auto i = 0;

will give you

int i = 0;
